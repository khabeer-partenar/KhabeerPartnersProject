# Khabeer Partners Project


To install this project do the following steps:

Setting Commands:
-
- cp .env.example .env
- composer install
- php artisan key:gen
- set values in .env file
    (app_url, mysql, mail, mobily, help desk email)
- php artisan migrate
- php artisan passport:install
- php artisan db:seed
- php artisan storage:link
- npm install
- npm run prod
